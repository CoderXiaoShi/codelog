# codesave

码记: 程序员专用的笔记软件

---

基于 git 仓库, 优点: 私密, 多端同步, 可以当做文档网盘使用

### 功能列表

- [x] 自动同步
- [x] 立即同步
- [x] 打开 vsCode
- [x] [开启|关闭] 自动同步
- 开机自动启动
  借助 pm2 实现

- 多仓库管理
  借助 pm2 实现
- 会记住 每一个需要同步的文件目录地址, 会批量的同步这些项目
- 全局保存 现有的笔记仓库

- 修改同步频率

v2.0

```
  进入任何目录先判断是否为 git 仓库
    如果是 直接定位到 该仓库, 从 pm2 中调出正在执行的进程
    如果不是则展示 当前的 仓库列表, 用方向键选择
    - 仓库1
    - 仓库2
    - 仓库3
```
