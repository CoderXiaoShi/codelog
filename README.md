# 码记

![NPM version](https://img.shields.io/npm/v/codesave?label=codesave&logo=npm&style=for-the-badge)

码记: 程序员专用的笔记软件, 它把 git 仓库当做存储笔记的地方. 并且固定时间自动同步 git 仓库

---

基于 git 仓库, 优点: 私密, 多端同步, 可以当做文档网盘使用

## 功能概述

- 自动同步
- 立即同步
- 打开 vsCode
- [开启|关闭] 自动同步

## 安装

建议使用 npm 全局安装

    npm install -g codesave

然后到你的 git 仓库中运行

    $ cd /仓库地址/
    $ codesave

看到这个输出就证明启动成功了

```
---------------------
欢迎使用 [码记]
---------------------

1. 开始自动同步 [开启]
2. 立刻同步代码

5. 帮助
6. 退出

同步失败

请继续输入:
```
